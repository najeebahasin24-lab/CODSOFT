# CODSOFT
Machine Learning Internship Projects-CODSOFT

Hi! 👋

This repository contains the Machine Learning projects that I completed during my internship at CODSOFT. Each project helped me understand different machine learning concepts such as data preprocessing, model training, prediction, and performance evaluation.

## Projects

### 1. Spam SMS Detection
- Built a model to classify SMS messages as Spam or Ham.
- Used TF-IDF Vectorization and Multinomial Naive Bayes.
- Achieved around 96% accuracy.

### 2. Customer Churn Prediction
- Developed a model to predict whether a customer is likely to leave a company.
- Used Random Forest Classifier.
- Performed data preprocessing, model training, and evaluation.

### 3. Credit Card Fraud Detection
- Built a machine learning model to detect fraudulent credit card transactions.
- Used Random Forest Classifier.
- Evaluated the model using Accuracy, Confusion Matrix, and ROC Curve.

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
