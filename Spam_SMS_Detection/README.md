# Spam SMS Detection

This project detects whether an SMS message is Spam or Ham using Machine Learning.

## Algorithm Used
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Features
- Data preprocessing
- Text vectorization
- Spam/Ham prediction
- Model evaluation using Accuracy, Confusion Matrix and ROC Curve

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib

**Accuracy:** Approximately 96.86%
